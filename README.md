# Token Action HUD - ACKS

Token Action HUD support for the Adventurer Conqueror King System (ACKS) in Foundry VTT.

## Features

- Attacks (weapons)
- Spells (sheet access)
- Checks (STR, INT, WIL, etc.)
- Saves
- Adventuring skills
- Initiative
- Clean HUD layout (no nested ACKS tab)

## Requirements

- Foundry VTT v13+
- Token Action HUD Core
- ACKS system

## Known Issues

- Spell casting currently opens the sheet (ACKS upstream issue)

## Installation

Install via manifest URL (see Releases).
