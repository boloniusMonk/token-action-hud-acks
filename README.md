# Token Action HUD - ACKS

Token Action HUD support for the Adventurer Conqueror King System (ACKS) in Foundry VTT.

## Features

- Attacks (weapons)
- Spells
- Checks (STR, INT, WIL, etc.)
- Saves
- Adventuring skills
- Proficiencies (rollable)
- Initiative

## Requirements

- Foundry VTT v13+
- Token Action HUD Core
- ACKS system

## Installation

- On Foundry VTT's Configuration and Setup screen, go to Add-on Modules
- Click Install Module
- In the Manifest URL field, paste: https://github.com/boloniusMonk/token-action-hud-acks/releases/latest/download/module.json
- Click Install next to the pasted Manifest URL
